RTMP Server

纯PHP自研轻量级RTMP直播服务｜零FFmpeg、Nginx第三方流媒体依赖，私有化直播底层基础设施

PHP >= 8.1 CLI RTMP / HTTP-FLV / WS-FLV HLS / fMP4 / MP4录制 Linux epoll 高并发
GitHub 仓库 配套工具 flv2mp4

项目核心亮点

完全零第三方依赖

不依赖 FFmpeg、Nginx、Swoole 等流媒体组件，纯PHP手写完整RTMP协议栈，开箱即用。

全协议全覆盖

原生支持 RTMP、HTTP-FLV、WebSocket-FLV、HLS-TS、fMP4 分片，PC/手机/微信全平台播放。

跨平台兼容

Linux自动启用epoll事件驱动高并发；Windows自动降级select，本地调试无门槛。

自动录制转码

并行FLV原始流录制、实时HLS/fMP4切片，直播结束自动合并完整MP4点播文件。

多进程IPC流同步

自研TCP进程间通信，多核CPU充分利用，多Worker进程共享直播流，支持大规模并发。

集群分布式部署

内置FLV分发网关、静态文件网关，支持多级级联、负载均衡，可支撑十万级在线观众。

内置Web推拉流页面

自带屏幕采集推流、本地文件循环推流、多码率弱网适配、多路画面合并页面，无需前端二次开发。

推流鉴权防护

支持stream key密钥鉴权，拦截非法推流覆盖直播间，保障私有直播安全。

快速启动

1. 安装项目

composer create-project xiaosongshu/rtmp_server

2. 启动主服务

php server.php

启动后自动监听：1935(RTMP)、8501(FLV/WS-FLV)、80(静态网页/点播)

3. OBS 推流地址示例

rtmp://127.0.0.1:1935/live/stream

4. 浏览器直接观看直播

http://127.0.0.1/index.html
